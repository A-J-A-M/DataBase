This program was made with the Oracle SQL 12c database.

In the file OracleSQL.java you can change the port if needed

In the file Prb.java, you must change the SID to your SID, with the
respective username and password you use to create the tables.

You should run the file tablas.txt in SQL or copy everything there to
create and poblate the tables.

Use Prb.java to run the application.
